# 🛒 GreenCart - MERN Stack E-Commerce Platform

GreenCart is a modern, fully functional e-commerce web application built using the MERN stack (MongoDB, Express.js, React, Node.js). It features seamless image management via Cloudinary and a secure checkout flow powered by Stripe.

🔗 **[Live Demo](https://greencart-eight-steel.vercel.app/)**

## 🚀 Features

- **User Authentication:** Secure signup, login, and JWT-based session handling.
- **Product Management:** Complete CRUD operations for products (Admin dashboard functionality).
- **Dynamic Image Uploads:** Powered by **Cloudinary** for scalable, fast image hosting and processing.
- **Shopping Cart & Checkout:** Intuitive add-to-cart workflow with real-time total calculation.
- **Secure Payments:** Full **Stripe API** integration supporting secure credit/debit card transactions.
- **Order Tracking:** Order history management and order success/failure tracking.
- **Responsive Design:** Optimized for mobile, tablet, and desktop viewing.

---

## 🛠️ Tech Stack

### Frontend
- **React.js** (Hooks, Context API/Redux for State Management)
- **React Router DOM** (Client-side routing)
- **Axios** (API requests)
- **Tailwind CSS / Bootstrap** (Styling)

### Backend
- **Node.js** & **Express.js** (RESTful API architecture)
- **MongoDB** & **Mongoose** (Database & object data modeling)
- **JWT (JSON Web Tokens)** & **Bcrypt.js** (Authentication & password hashing)

### Third-Party Services
- **Stripe SDK** (Payment processing)
- **Cloudinary SDK** (Image storage)
- **Vercel** (Frontend Hosting)

## 📋 Prerequisites

Before you begin, ensure you have the following installed:
- [Node.js](https://nodejs.org/) (v16.x or higher)
- [MongoDB](https://www.mongodb.com/) (Local instance or MongoDB Atlas account)
- A [Stripe Developer Account](https://stripe.com/)
- A [Cloudinary Account](https://cloudinary.com/)
