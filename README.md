🛒 GreenCart – MERN Stack E-Commerce Application
📌 Overview

GreenCart is a full-stack e-commerce web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js). It provides a seamless shopping experience where users can browse products, add items to cart, and securely place orders.

🚀 Features
👤 User Features
User Registration & Login (JWT Authentication)
Browse Products by Category
Search & Filter Products
Add to Cart / Remove from Cart
Secure Checkout Process
Order History & Tracking
🛠️ Admin Features
Add / Update / Delete Products
Manage Orders
View Users
Dashboard Analytics (optional)
